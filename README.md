# Family Barbershop — Demo v2

Demo landing page pitch for Family Barbershop (Kissimmee, FL).
Built by **iLaunch U** to showcase a redesigned website with:

- 10 barbers, each with portrait + 4-cut portfolio + per-barber booking
- Black / red / blue color-blocked sections
- Modern barbershop interior hero with overlay
- Service menu, gallery, testimonials, location map, Instagram link
- Fully responsive (mobile → desktop)

## Live
Served via GitHub Pages — see repo settings.

## Stack
Static HTML/CSS/JS. Zero build step. Python http.server or any static host works.
